How To Use

使い方の流れ

試合当日、担当を決めてそれぞれのツールを開くだけ。
試合後はJSONを1ファイルにまとめてStreamlitへ。

01
担当を決める
5つのツールを4名以上で分担。スコアシート担当が中心となりQ管理も行う
02
スマホで記録
各自が担当ツールをスマホで開き、試合中リアルタイムで入力。タップ中心の直感的UI
03
JSONエクスポート
試合後、各ツールの「出力タブ」からJSONをダウンロード。Streamlit連携用のフォーマット
04
Streamlitで分析
ダッシュボードにJSONをアップロードするだけで自動集計・グラフ化が完了
Data Format

JSONフォーマット

全ツールのJSONは共通のmetaキーを持ち、 Streamlit側で自動識別できます。

📊 スコアシート / ターンオーバー
"meta": { "tool": "GameDataTool v1.0", "enemy": "○○大学", "date": "2025-04-01" }, "shots": [...], "turnovers": [...], "clearance": {...}
⏱ ポゼッション
"meta": { "tool": "PossessionTool v1.0" }, "of_possession": { "sets": [...], "by_q": [...] }, "clrd_possession": { "by_q": [...] }
🏃 GB・ファール
"meta": { "tool": "GBFoulTool v1.0" }, "gb": { "records": [...], "summary": {...} }, "fouls": { "records": [...] }
🥅 ゴーリー
"meta": { "tool": "GoalieTool v1.0" }, "shots": [ { "course": 0, "result": "save", "side": "kyoto" } ], "summary": {...}
